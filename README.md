# Query-Enabled Behaviour Tree for Job Shop Scheduling

## Setup

### Using pip

#### 1. Create and activate a virtual environment

```bash
python -m venv .venv

# Linux/macOS
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

#### 2. Install the local package
``` bash
pip install -e ./core
```

### Using uv
``` bash
uv sync
```

## Running
### Run tests
```bash
python test.py
```
